# Results

This folder is intended for storing reproducibility outputs generated by the AI-based mineral prospectivity mapping workflow.

The main outputs may include:

* validation performance metrics
* ROC curves
* AUC values
* precision, recall, F1-score, and accuracy values
* continuous prospectivity scores
* binary prospectivity classes
* five-class prospectivity maps
* GIS-ready prediction tables
* final model configuration tables

## Recommended structure

```text
results/
├── G1/
│   ├── metrics.csv
│   ├── roc_curve.png
│   ├── prospectivity_scores.csv
│   └── prospectivity_classes.csv
│
└── G2/
    ├── metrics.csv
    ├── roc_curve.png
    ├── prospectivity_scores.csv
    └── prospectivity_classes.csv
```

## Important note

The outputs generated from the public anonymized datasets are intended to verify the execution of the G1 and G2 pipelines and to support reproducibility of the machine-learning workflow under the anonymized data-release setting.

Because raw geospatial coordinates and original GIS layers are withheld for mineral-location protection, GIS-ready outputs generated from the public repository should not be interpreted as disclosure of the original study-area coordinate network.

The provided datasets, Python scripts, fixed random seeds, and `spatial_block_id` field allow reviewers and readers to inspect and rerun the modeling workflow, including spatial GroupKFold validation, multi-level stacking, performance evaluation, and generation of local output files.
