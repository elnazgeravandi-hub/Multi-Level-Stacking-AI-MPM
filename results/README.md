# Results

This folder is intended for storing reproducibility outputs generated by the AI-based mineral prospectivity mapping workflow.

The main outputs may include:

* validation performance metrics
* ROC curves
* AUC values
* precision, recall, F1-score, and accuracy values
* continuous prospectivity scores
* binary prospectivity classes
* five-class prospectivity maps
* GIS-ready prediction tables
* final model configuration tables

## Recommended structure

```text
results/
├── G1/
│   ├── metrics.csv
│   ├── roc_curve.png
│   ├── prospectivity_scores.csv
│   └── prospectivity_classes.csv
│
└── G2/
    ├── metrics.csv
    ├── roc_curve.png
    ├── prospectivity_scores.csv
    └── prospectivity_classes.csv
```

## Notes

Large output files are not necessarily included in this repository. Users can reproduce the results by running the provided Python scripts with the anonymized datasets available in the `data/` folder.

The `results/` folder is provided to clarify the expected output structure and improve workflow reproducibility.
