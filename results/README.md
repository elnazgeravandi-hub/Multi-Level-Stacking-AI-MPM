# Results folder

This folder is used for storing outputs generated by running the anonymized reproducibility workflow.

The outputs are generated locally when the Python scripts are executed.

These outputs are intended for reviewer verification and workflow reproducibility only.

They should not be interpreted as the final confidential prospectivity results reported in the manuscript.

## Expected outputs

After running the workflow, the following output files may be generated for each modeling group:

```text
results/
├── G1/
│   ├── MultiStack_G1_validation_predictions.csv
│   ├── MultiStack_G1_spatial_metrics.csv
│   ├── MultiStack_G1_classification_report.txt
│   ├── ROC_MultiStack_G1_spatial.png
│   ├── MultiStack_G1_Test_FullOutput_spatial.csv
│   └── MultiStack_G1_For_ArcMap_spatial.csv
└── G2/
    ├── MultiStack_G2_validation_predictions.csv
    ├── MultiStack_G2_spatial_metrics.csv
    ├── MultiStack_G2_classification_report.txt
    ├── ROC_MultiStack_G2_spatial.png
    ├── MultiStack_G2_Test_FullOutput_spatial.csv
    └── MultiStack_G2_For_ArcMap_spatial.csv
```

## Output interpretation

The generated outputs include:

* validation probabilities;
* spatial validation metrics;
* ROC curve;
* AUC value;
* accuracy, precision, recall, and F1-score;
* continuous prospectivity scores;
* binary prospectivity classes;
* five-class prospectivity classes;
* GIS-ready CSV tables.

These files reproduce the structure of the manuscript workflow outputs using anonymized demonstration datasets.

They do not disclose:

* original mineral occurrence coordinates;
* original non-occurrence coordinates;
* original coordinate network;
* original GIS layers;
* final confidential prospectivity maps;
* final confidential numerical results reported in the manuscript.

## Important note

The repository is designed to demonstrate that the Python workflow can be executed and that it generates outputs with the expected structure.

Because the public datasets are anonymized demonstration datasets, the numerical values generated from them may differ from the final manuscript results.
