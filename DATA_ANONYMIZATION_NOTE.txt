# DATA ANONYMIZATION NOTE

The datasets released in this repository are anonymized tabular modeling datasets prepared to support verification and reproducibility of the machine-learning workflow described in the manuscript.

Because the original geospatial datasets contain sensitive mineral occurrence and non-occurrence locations, the exact UTM coordinates, CRS information, UTM zone, coordinate origin, raw occurrence/non-occurrence coordinates, original spatial block codes, and raw GIS layers are not publicly released.

The released x and y fields are anonymized relative local coordinates. They do not represent real-world coordinates, UTM coordinates, or the original study-area coordinate network.

Spatial validation is preserved using the provided spatial_block_id field. This field is an anonymized grouping identifier used by the Python scripts for deterministic spatial block splitting and GroupKFold validation.

The spatial_block_id field must not be used as a predictor variable.

The public datasets allow reviewers and readers to run the G1 and G2 workflows, inspect the multi-level stacking implementation, reproduce the code-execution procedure, and generate local outputs without disclosing sensitive mineral-location information.
